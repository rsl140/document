> 后进先出的数据结构，可以利用js中的数组实现(push & pop)

![20210609150113](https://cdn.jsdelivr.net/gh/rsl140/imgCdn@main/imgs/20210609150113.png)

``` js
const stack = []

// 入栈
stack.push(1)
stack.push(2)

// 出栈
const item = stack.pop()
```
